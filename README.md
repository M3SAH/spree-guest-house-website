# Spree Guest House Website

A modern luxury hospitality website designed and developed for Spree Guest House, located in Kansenshi, Ndola, Zambia.

The website was built to present the guest house with a premium online presence focused on:
- accommodation
- conferencing
- events
- hospitality services
- restaurant services
- bookings and inquiries

## Features

- Multi-page luxury hospitality design
- Fully responsive layout
- Premium editorial-style UI
- Accommodation showcase
- Events & conferencing sections
- Restaurant and dining presentation
- Mobile-friendly navigation
- WhatsApp contact integration
- Smooth animations and transitions
- Optimized image presentation
- Elegant booking and inquiry experience

## Pages

- Home
- Rooms
- Room Details
- Events & Conferencing
- Dining
- Gallery
- About
- Contact
- Booking
- FAQ
